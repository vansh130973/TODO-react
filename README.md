# TODO App

A modern TODO application with authentication, built using React 18, Vite, and Bootstrap 5. All data persists in localStorage.

## Features
### Authentication
- User registration
- User login & logout
- Separate todo lists per user
- Session persistence using `localStorage`

### Todo Management
- Add new todos
- Edit existing todos (inline editing)
- Delete todos
- Auto timestamp for each todo

---

## How to Run the Project

## Clone and install
- git clone https://github.com/vansh130973/TODO-react.git
- cd TODO-react
- npm install
- Run with this Command:- npm run dev

---

## How the App Works

- Register with username and password
- Login to access your private todo list
- Add todos with auto-timestamps
- Edit or delete todos as needed
- Logout when done

## Technologies Used

- React 18
- Vite 5
- Bootstrap 5
- localStorage API
- Custom React Hooks
- ES6 Modules
